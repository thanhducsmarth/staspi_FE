TỔNG HỢP KÍCH THƯỚC CHUẨN HÌNH ẢNH TRÊN CÁC KÊNH SOCIAL MEDIA

9/9/2022

admin

#Admin_Linh

  1. Instagram

    2. Có một số định dạng hình ảnh chuyên biệt dành riêng cho Instagram. Stories, highlights, carousel ads hay IGTV đều sở hữu những kích thước riêng.

    Kích thước Instagram

    Profile picture: 320 x 320px

    Post & image ads:

    • Landscape: 1080 x 566px
    • Portrait: 1080 x 1350px
    • Square: 1080 x 1080px
    • Recommended width: 1080px
    • Recommended height: 566-1350px

    Thumbnails:

    • Display size: 161 x 161px
    • Recommended width: 1080px

    Stories & Stories ads:

    • Recommended size: 1080 x 1920px
    • Aspect ratio: 9:16

    Carousel:

    • Landscape: 1080 x 566px
    • Portrait: 1080 x 1350px
    • Square: 1080 x 1080px
    • Recommended width: 1080px
    • Recommended height: 566-1350px

    IGTV:

    • Size: 1920 x 1080px
    • Cover image height: 420px
    • Cover image width: 654px
  2. Facebook

    3. Facebook cũng có tiêu chí hình ảnh tương tự như Instagram. Facebook tập trung nhiều vào posts, banners và stories.

    Kích thước chuẩn Facebook

    Fanpage:

    • Profile picture: 170 x 170px trên máy tính, 128 x 128px trên điện thoại
    • Cover photo: 820 x 312px trên máy tính, 640x360px trên điện thoại
    • Cover video: 820 x 462px
    • Posts: 1200 x 630px

    Carousel (2-10 hình): 1200 x 1200px

    Event cover photos: 1200 x 628px

    Facebook stories: 1080 x 1920px

    Facebook ads: 1080 x 1080px

  3. Twitter
    4. Kích thước chuẩn trên Twitter

    Profile photos: 400 x 400

    • Minimum image size: 200 x 200 pixels
    • Maximum file size: 2MB

    Header photo size: 1500 x 1500 pixels

    In-stream photos: 1600 x 900 pixels

    Supported formats: JPG, PNG và GIF

  4. Tiktok

    5. Tiktok là nền tảng đăng tải video được nhiều người sử dụng hiện nay. Định dạng trên tiktok khá đơn giản, bạn cần quan tâm tới kích thước video, hình ảnh hồ sơ và quảng cáo để đạt hiệu quả tốt hơn.

    Kích thước video Tiktok

    Tiktok profile picture: 200 x 200px

    Tiktok video: 1080 x 1920px

    Tiktok ad:

    • Portrait: 720 x 1280px
    • Square: 1200 x 1200px
    • Landscape: 1280 x 720px
  5. Youtube

    6. Video chiếm vị trí hàng đầu đối với kênh Youtube, điều đó có nghĩa là bạn chỉ cần điều chỉnh kích thước hình ảnh phù hợp để góp phần làm cho kênh trở nên bắt mắt hơn.

    Kích thước chuẩn trên Youtube

    Page:

    • Profile picture: 800 x 800px
    • Banner: 2560 x 1440px
    • Video: 1920 x 1080px

    Youtube display ads: 300 x 250px

    Youtbe overlay ads: 480 x 60px

  6. Pinterest

    7. Pinterest là nền tảng duy nhất cho phép người dùng tìm kiếm bằng hình ảnh. Pinterest cũng được biết đến là nơi chia sẻ những hình ảnh chất lượng cao được nhiều người yêu thích. Do đó, việc đầu tư vào hình ảnh và kích thước chuẩn là yếu tố tiên quyết cần có.

    Kích thước Pinterest

    Standard pin: 1000 x 1500px

    Square pin: 1000 x 1000px

    Long pin: 1000 x 2100px

    Infographic: 1000 x 3000px

    Trên đây là tổng hợp kích thước chuẩn hình ảnh cho từng trang mạng xã hội tương ứng, thật khó để nhớ hết những thông số và kích thước hình ảnh này. Do đó, bạn đừng quên lưu lại bài đăng này để tham khảo khi cần để tối ưu hóa hình ảnh trên mọi nền tảng từ đó đạt được mục tiêu đề ra.
